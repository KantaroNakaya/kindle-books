## Git 管理のガイドライン

### ✅ 管理するファイル

-   `kindle-book-outline.md` - 構成案
-   `kindle-book-content.md` - 執筆内容
-   `.gitignore` - Git 除外設定
-   `README.md` - プロジェクト説明

### ❌ 管理しないファイル

-   スクリーンショット（容量が大きい）
-   生成されるファイル（.epub, .mobi 等）
-   一時ファイル
-   エディタ設定ファイル

## コミットメッセージの例

```
feat: 第1章の執筆完了
feat: 第2章の構成を追加
fix: 第3章の誤字修正
```
